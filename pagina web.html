¿Qué es la programación? 💻

Programar es darle instrucciones a una computadora para que haga algo. Como una receta de cocina — tú escribes los pasos y la computadora los sigue exactamente.

📦

Variables

Guardan información para usarla después.

// JavaScript
let nombre = "Juan"
let edad = 20

# Python
nombre = "Juan"
edad = 20
🔀

Condicionales

Toman decisiones según una condición.

// JavaScript
if (edad >= 18) {
  console.log("Mayor de edad")
}

# Python
if edad >= 18:
    print("Mayor de edad")
🔁

Bucles

Repiten una acción varias veces.

// JavaScript
for (let i = 0; i < 5; i++) {
  console.log(i)
}

# Python
for i in range(5):
    print(i)
⚙️

Funciones

Bloques de código reutilizables.

// JavaScript
function saludar(nombre) {
  return `Hola, ${nombre}!`
}

# Python
def saludar(nombre):
    return f"Hola, {nombre}!"
📋

Listas

Guardan varios valores en una sola variable.

// JavaScript
let frutas = ["manzana", "pera"]

# Python
frutas = ["manzana", "pera"]
🗂️

Objetos

Guardan datos con nombre y valor.

// JavaScript
let persona = {
  nombre: "Juan",
  edad: 20
}

# Python
persona = {
  "nombre": "Juan",
  "edad": 20
}
